Όταν η FinTrack μας προσέγγισε, είχαν ένα λειτουργικό προϊόν αλλά όχι ένα αγαπημένο. Η πλατφόρμα αναλυτικών τους λειτουργούσε αρκετά καλά για τους προχωρημένους χρήστες που ήταν πρόθυμοι να πλοηγηθούν στην πολυπλοκότητά της, αλλά έχανε λιγότερο τεχνικούς χρήστες που απλά ήθελαν να κατανοήσουν την οικονομική τους κατάσταση με μια ματιά. Η πρόκληση ήταν σαφής: να κάνουμε τα εξελιγμένα οικονομικά αναλυτικά να φαίνονται αβίαστα χωρίς να υποβαθμίσουμε τις δυνατότητες στις οποίες βασίζονταν οι προχωρημένοι χρήστες.

Ξεκινήσαμε χαρτογραφώντας κάθε διαδρομή χρήστη, εντοπίζοντας τις κρίσιμες διαδρομές που αντιπροσώπευαν το 80% των καθημερινών αλληλεπιδράσεων. Από εκεί, σχεδιάσαμε ένα σύστημα προοδευτικής αποκάλυψης — η προεπιλεγμένη προβολή παρουσιάζει μια καθαρή, σαρώσιμη επισκόπηση βασικών μετρικών, ενώ τα προχωρημένα χειριστήρια και οι δυνατότητες ανάλυσης σε βάθος αποκαλύπτονται σταδιακά καθώς οι χρήστες εμβαθύνουν. Η αρχιτεκτονική βασισμένη σε widgets σημαίνει ότι δεν χρειάζεται να μοιάζουν δύο πίνακες ελέγχου· ένας CFO που εστιάζει στις ταμειακές ροές βλέπει ένα θεμελιωδώς διαφορετικό layout από έναν αναλυτή που παρακολουθεί κατηγορίες εξόδων, ωστόσο και οι δύο διεπαφές αισθάνονται συνεκτικές και διαισθητικές.

Από τη μηχανολογική πλευρά, η απόδοση ήταν υψίστης σημασίας. Οι επαγγελματίες του χρηματοοικονομικού τομέα αναμένουν στιγμιαίες αποκρίσεις, και οποιαδήποτε καθυστέρηση στην απεικόνιση δεδομένων υπονομεύει την εμπιστοσύνη στους ίδιους τους αριθμούς. Δημιουργήσαμε ένα προσαρμοσμένο επίπεδο συγκέντρωσης δεδομένων σε Node.js που προυπολογίζει τα συνήθη ερωτήματα και αποθηκεύει τα αποτελέσματα έξυπνα στην cache, μειώνοντας τους μέσους χρόνους φόρτωσης του πίνακα ελέγχου κάτω από 800 χιλιοστά του δευτερολέπτου ακόμα και με σύνολα δεδομένων που εκτείνονται σε εκατομμύρια συναλλαγές. Οι οπτικοποιήσεις D3.js αποδίδονται προοδευτικά, ώστε οι χρήστες να βλέπουν ουσιαστικά δεδομένα εντός των πρώτων 200 χιλιοστών του δευτερολέπτου ενώ πιο σύνθετα γραφήματα ολοκληρώνονται στο παρασκήνιο.
